===========================================
GitHub Setup Instructions for SysOps
===========================================

Your SysOps framework is ready to push to GitHub!

OPTION 1: Install GitHub CLI (Recommended)
-------------------------------------------
1. Install GitHub CLI:
   macOS: brew install gh

2. Authenticate:
   gh auth login

3. Run the setup script:
   ./scripts/setup-github.sh


OPTION 2: Manual Setup (If you don't want to install gh CLI)
-------------------------------------------------------------
1. Go to https://github.com/PIPERSEC and create a new repository:
   - Name: sysops
   - Description: Systems Engineering Framework with AI-assisted workflows
   - Visibility: Your choice (public or private)
   - DO NOT initialize with README, .gitignore, or license

2. After creating, run these commands:
   
   git remote add origin https://github.com/PIPERSEC/sysops.git
   git branch -M main
   git push -u origin main

   OR if you prefer SSH:
   
   git remote add origin git@github.com:PIPERSEC/sysops.git
   git branch -M main
   git push -u origin main

3. Visit your repository to verify: https://github.com/PIPERSEC/sysops
