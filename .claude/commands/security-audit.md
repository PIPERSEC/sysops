# Security Audit

Perform a comprehensive security audit using the security-review CoT template.

Review:
1. Authentication and authorization mechanisms
2. Data protection (encryption, secrets management)
3. Network security (segmentation, firewalls, ACLs)
4. Logging and monitoring for security events
5. Compliance with relevant standards
6. Known vulnerabilities and patches
7. Access control and privilege management

Provide a detailed report with:
- Critical findings requiring immediate action
- High priority issues (30 days)
- Medium priority improvements (90 days)
- Long-term security enhancements

Save the audit report in `docs/security/audit-YYYYMMDD.md`
