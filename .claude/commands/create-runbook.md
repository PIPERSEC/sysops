# Create Runbook

Create a comprehensive runbook for the specified operation or service.

The runbook should include:

1. **Overview**: Purpose and scope
2. **Prerequisites**: Required access, tools, and knowledge
3. **Step-by-Step Procedures**: Detailed instructions
4. **Troubleshooting**: Common issues and solutions
5. **Rollback Procedures**: How to undo changes if needed
6. **Verification**: How to confirm success
7. **Related Documentation**: Links to relevant resources

Save the runbook in `docs/runbooks/` with a descriptive filename.
